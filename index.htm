syncped is a SYNChronizing Portable EDitor supporting more than 100 programming languages, is available under Windows and Linux, has a vi compatibility mode, and can be configured to work with most version control systems.

Currently translation to Dutch and French are available, other languages are easy to implement, translating two po files is all that is necessary.

syncped version 3.0.0 is released.
